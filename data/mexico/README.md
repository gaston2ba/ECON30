# Mexico City data

| Path | Description |
|------|-------------|
| `workbook/DataMexCity.xlsx` | INEGI-based alcaldía panel (1860–2020): population, density, transit, parks, housing, third spaces |
| `sources/index-mex-city.zip` | Original component spreadsheets and documentation |
| `historical/mexico-city-c1895.csv` | Porfirian occupational baseline (~1895–1900) for Figure 1 |
| `exports/` | Generated scores (run `python scripts/export_mexico_data.py`) |

**Income note:** The workbook uses **persons per dwelling** as a crowding proxy until ENIGH median income is added.
